Proyecto

Agente de IA

Un agente desarrollado como parte de una tesis, pensado para automatizar tareas, centralizar información y responder dentro de un flujo conversacional.

Volver a proyectos

Contexto

Trabajo con restricciones reales

El proyecto se construyó para un entorno profesional, así que la información detallada no puede mostrarse completa. Aun así, la solución integra automatización y asistentes para tareas concretas.

Base técnica

n8n como núcleo

La lógica principal se montó sobre flujos de n8n, combinando web scraping, RAG y servicios externos para coordinar respuestas y automatizaciones.

Integraciones

Conectado a herramientas reales

Se integraron Google Drive, Gmail, Calendario, Hojas de cálculo, Supabase y WhatsApp mediante EvolutionAPI, todo desplegado en un VPS.

Qué resuelve

El objetivo fue reducir trabajo manual y reunir varias fuentes de información en un solo flujo operativo. La experiencia final se pensó para ser útil, rápida y fácil de mantener.

Más que un demo aislado, este proyecto muestra integración entre automatización, datos y mensajería en un entorno que simula una necesidad real de negocio.
