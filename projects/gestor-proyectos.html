Contexto

Proyecto académico completo

Fue creado para una evaluación de la carrera como una solución de gestión con front y backend, pensando en una experiencia funcional y ordenada.

Tecnología

.NET y SQL Server

La solución se desarrolló en C# con ASP.NET Core, integrando una API interna, Entity Framework y una base de datos SQL Server.

Valor

Pensado para uso real

El foco estuvo en resolver un flujo de trabajo simple pero útil: administrar tareas, visualizar estados y mantener trazabilidad sin complicar la interfaz.

Lo que incorpora

El proyecto reúne autenticación, módulos administrativos, vistas MVC y un tablero Kanban para visualizar el avance de forma simple.

También contempla catálogos, reportes y administración de información base, lo suficiente para sentirse como una aplicación completa dentro de un entorno académico.
